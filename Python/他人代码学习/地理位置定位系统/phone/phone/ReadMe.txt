1、各个程序文件的基本功能
（1）hashmap1.py
基于haspmap的数据结构，存储中国手机号段（前7位）对应的地理位置，采用链地址法解决hash冲突
中国固定电话区号段及其对应的地理位置，因为内容较少（300多条），直接用列表存
phone.txt:存手机号段及对应的地理位置
telphone.txt:存固定电话区号段及对应的地理位置

（2）format_phone.py
基于正则表达式，处理多种格式的电话，使之统一格式化

（3）qianduan_format.py
因为（1）中的手机号及地理位置和固话区号及地理位置是从网上爬取的中国号段对应的地理位置，可能与
咱实验室前端需展示的规定的各个省份的命名不大一样，比如湖南的‘湘西土家族苗族’，其标准输出应该是‘湘西土家族苗族自治区’，
所以此程序主要与前端统一各个省市的命名

###主程序
（4）get_resu.py
输入一个电话，以元组返回这个电话的 标准格式、省份、城市、类型、是否中国（1表示中国，0不是，但有些是中国的号码，可能国际代码处写的国外的）
类型：1为正常解析，2为异常解析（如位数不对或者是有些补0有些去0才解析成功），3位解析不了的电话
